# report

Reports useful version numbers to copy into the Nx issue template.

## Usage

```shell
nxcc report
```

Install `nxcc` globally to invoke the command directly using `nxcc`, or use `npx nxcc`, `yarn nxcc`, or `pnpm nxcc`.
