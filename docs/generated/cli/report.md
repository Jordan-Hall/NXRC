# report

Reports useful version numbers to copy into the Nx issue template.

## Usage

```shell
nxrc report
```

Install `nxrc` globally to invoke the command directly using `nxrc`, or use `npx nxrc`, `yarn nxrc`, or `pnpm nxrc`.
